Parágrafos e quebras de linhas

Você pode escrever um parágrafo de qualquer maneira, basta colocar entre as estruturas p e /p. A formatação se manterá como se fosse um paráfrago comum, mesmo que dentro do código html não esteja.

Se precisar quebrar linha dentro de um parágrafo, usar a tag br.
